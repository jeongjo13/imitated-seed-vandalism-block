# 기본 사용 방법
코드 중간에 wiki_username 변수, wiki_password 변수 정의하는 곳에 자신의 아이디와 비번 넣으시고 나서 main.py 실행하시면 됩니다.

# 추가 사용 방법
* 반달성 키워드 추가를 하고 싶으시면 코드 중간에 vandalism 리스트에 키워드 추가하시면 됩니다.
* 예외 처리되는 사용자를 수정하시려면 blocked 리스트에 사용자명을 추가하시면 됩니다.
